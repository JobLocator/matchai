# matchai

AI-powered job matching platform.

## Structure

| Folder | Stack | Purpose |
|--------|-------|---------|
| `/frontend` | Next.js 14, Tailwind, Clerk | User interface |
| `/backend` | Node.js, Fastify, Prisma | REST API |
| `/crawler` | Python, Playwright | Job ingestion |

## Getting started

1. Copy `.env.example` to `.env` and fill in your keys
2. See each folder's README for setup instructions
